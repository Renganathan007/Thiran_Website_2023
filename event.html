CODING HOUSE

EVENT DESCRIPTION :

  Hey geeks!! Ready to walk into the coding house? Solve a puzzle to find the key to the next. If you win, you walk out. Lose and you get haunted. Have fun with games and coding… coding house awaits you!!!

ROUND-WISE DESCRIPTION :

Round 1 : Blind Coding
  A simple problem will be given and competitors will be requested to write a code for that problem with their monitor off. Competitors with less number of errors or no errors will be moved on to the next round.

Round 2 : Debug the code
  In the second round, a code with some errors will be provided to the competitors and the competitors are expected to decode the errors. The teams who can decode all the errors or most of the errors will be moved to the next round.

Round 3 : Ultimate puzzle
  A problem statement will be given. Competitors can solve the problems by earning clues. One clue will be provided if the competitors solve the given puzzle or challenge. One who earns all the clues and completes the problem first will be the winner.

EVENTS RULES :

  • Each team may contain 2 to 3 members.
  • Top-performing teams in the prelims will be selected for finals.
  • Try to bring Pens or papers if required.
  • The decision of the judge is final.
